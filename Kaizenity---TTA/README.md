# Kaizenity - TTA

Совместный проект для работы с коллегами.

## 📁 Структура проекта

```
Kaizenity---TTA/
├── 📁 shared/           # Совместные файлы
│   ├── 📁 projects/     # Проекты
│   ├── 📁 docs/         # Документация
│   └── 📁 resources/    # Ресурсы
├── 📁 personal/         # Личные файлы
│   ├── 📁 notes/        # Заметки
│   └── 📁 drafts/       # Черновики
├── 📁 templates/        # Шаблоны
└── 📁 archive/          # Архив
```
